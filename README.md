# Website LLC — Y.O.U. (You're Own Vision)

## What This Repository Is
This repository is the operating blueprint for a research-first website studio focused on small businesses that either have no website or have a weak, outdated, low-trust web presence.

The business model is simple:
1. Identify real businesses that need a stronger digital presence.
2. Qualify whether they are a good fit.
3. Build a structured research pack for each business.
4. Turn that pack into a website concept and sales-ready proposal.
5. Secure owner approval before any live publication.

## Qualification Standard
A business is a good fit if most of the following are true:
- no dedicated official website exists, or the current website is outdated or weak
- active and legitimate business
- enough public information exists to build a strong starter draft
- clear upside from trust / visibility improvements

Directories and listing pages do not automatically count as an owned website.
