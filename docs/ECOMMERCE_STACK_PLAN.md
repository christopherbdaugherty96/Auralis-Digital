# Ecommerce Stack Plan

Last updated: 2026-06-05

This document defines the practical ecommerce stack for Auralis Digital and its product channels.

## Stack Summary

```text
Shopify
|-- Lucid Creations collection
|-- RJ Print collection
|-- Printify / POD products
|-- curated dropshipping
|-- affiliate recommendations
|-- Shopify Email
|-- Judge.me reviews
`-- TikTok + Instagram traffic
```

## Platform

Use Shopify as the center of the ecommerce system.

Shopify should handle:

- products
- checkout
- payments
- taxes
- shipping settings
- orders
- customer records
- app connections
- sales channels

Auralis Digital can display products and route buyers to Shopify, but it should not process product checkout directly at this stage.

## Current Store Structure

Use one Shopify store for now.

Separate product lines through collections and Auralis Digital navigation:

```text
Shop
|-- Lucid Creations
`-- RJ Print
```

This keeps checkout, policies, email, analytics, order management, and store admin centralized while giving customers two clear shopping paths.

Do not create a second Shopify store yet. A separate RJ Print Shopify store is a later option only if the 3D printed product line grows enough to justify the extra maintenance.

## Lucid Creations

Lucid Creations is the art/POD/custom-design-aligned product line.

Use it for:

- apparel
- blankets
- pillows
- wall art
- home decor
- art-based products
- custom design-aligned products
- Printify / print-on-demand products

Customer-facing role:

```text
Art-driven products and custom creations.
```

## Print-On-Demand

Use Printify for products Auralis does not make in-house.

Good Printify categories:

- shirts
- hoodies
- blankets
- pillows
- mugs
- posters
- journals
- custom artwork products

Printify should stay operationally separate from RJ Print because it uses outside production and fulfillment.

## RJ Print

Use RJ Print for in-house or custom 3D print products.

Good RJ Print categories:

- koozies
- wallets
- lamps
- clocks
- chapstick holders
- accessories
- custom 3D prints
- small decor pieces
- made-to-order gift items

RJ Print may become the highest-margin product line because Auralis controls design, production, pricing, bundling, and custom work.

Customer-facing role:

```text
Original 3D printed products and functional designs.
```

## Dropshipping

Use one supplier app first, not several.

Recommended first options:

- Spocket
- Zendrop

Best-fit dropshipping categories:

- creative desk accessories
- LED lamps
- display stands
- home decor
- gift items
- small creator tools
- 3D-print-adjacent accessories

Customer-facing language should be curated and brand-aligned. Do not call a collection "Dropshipping" on the public site.

Potential public collection names:

- Creative Finds
- Curated Essentials

## Wholesale

Use Faire later, after demand is proven.

Wholesale can improve margins, but it adds inventory risk. It should not be used as the first testing layer.

## Email

Start with Shopify Email.

Move to Klaviyo only after there is real traffic, consistent sales, and a reason to need deeper automation.

## Reviews

Use Judge.me or Shopify-native review tooling.

Reviews matter most for:

- RJ Print
- custom products
- dropshipped products
- higher-priced products

Do not add fake reviews or imply unverified proof.

## Traffic

Start with TikTok and Instagram.

Best content types:

- 3D printing process videos
- product demos
- before/after custom designs
- packaging videos
- short product explanation videos
- creator desk or home decor setups

## Build Order

1. Keep one Shopify store.
2. Create or verify separate Shopify collections for Lucid Creations and RJ Print.
3. Add Auralis Digital navigation/shop paths so customers can browse Lucid Creations separately from RJ Print.
4. Finish RJ Print / 3D print planning.
5. Clean up and verify current Printify products.
6. Add reviews.
7. Set up Shopify Email.
8. Add affiliate recommendations.
9. Add one dropshipping supplier.
10. Connect and test TikTok and Instagram selling.
11. Use Faire only after demand is proven.

## Main Rule

Do not add products just because they are available.

Only add products that fit Auralis Digital, Lucid Creations, RJ Print, creativity, gifts, home decor, creator tools, custom design, or symbolic artwork.
