# Operating Rhythm & North Stars

Adopted: 2026-07-04. The systems phase is complete; systems now serve the business. This page is the filter for how time gets spent and what work gets accepted.

## North-star metrics

Every decision should increase one of these or reduce the time it takes to do so:

1. **Revenue** — total monthly sales
2. **Proof** — customer photos, reviews, repeat customers
3. **Audience** — people who regularly see the work

Automation, documentation, scripts, and dashboards are only justified by their effect on these three. If a proposed task doesn't move one, it's secondary.

## Weekly cadence

| Day | Mode | Work |
|---|---|---|
| Monday (1–2 hrs) | Operations | Review `BUSINESS_HEALTH.md`; fix drift/broken links; process orders; check Google/TikTok status; commit + push production-ready changes |
| Tue–Thu (80% of time) | Creation | Make pieces; photograph products; film short-form content; add products (follow `NEW_PRODUCT_CHECKLIST.md`) |
| Friday | Marketing | Publish content; outreach; ask for reviews; analyze what performed |

Maintenance stays contained to Monday. If it leaks into creation days, that's a signal to automate or drop it.

## Ecosystem rule

The branches reinforce each other — Lucid Creations (original art/products), RJ Print (in-house manufacturing), Auralis Digital (cash-flow services), and the agent tooling that lowers the cost of running all three. **Do not add a fifth branch until the existing four produce consistent results.**

## 90-day success definition (through ~2026-10-01)

- More verified sales (baseline: 1)
- A library of authentic customer proof (baseline: 0 reviews/photos)
- Manual maintenance replaced by automation only where it directly supports the first two

Everything else is secondary until these move consistently.
