# Assets — Build Your First Shopify Store

This folder holds the visual and downloadable assets for the **Build Your First Shopify Store** guide
(*Stand on Your Own™ · Guide 1*).

## What goes here

- **Lucid Creations screenshots** — real screenshots from the live store, used as worked examples.
- **Diagrams** — the fulfillment decision tree and any other explanatory graphics.
- **Worksheet graphics** — designed/printable versions of the guide's worksheets.
- **Final PDF assets** — cover art, section dividers, and exported PDF files.

## Rules

- **Do not add fabricated or placeholder screenshots.** Only real captures from actual stores/tools belong here.
- Until a real asset exists, the manuscript uses a visible text placeholder (for example, `[INSERT SCREENSHOT]`
  or `[INSERT LUCID CREATIONS EXAMPLE]`). Replace the placeholder only when the real asset is added here.
- Keep filenames descriptive, e.g. `ch04-shopify-store-details.png`, `ch07-fulfillment-decision-tree.png`.
- Do not commit private data, account emails, API keys, or anything that should not be public in screenshots.
