# Auralis Digital Shop Structure

This document records the intended shop/category structure before starting dedicated build work on the `LastStopShop` repository.

## Main Structure

```text
Shop
├── Partnered Creators
├── Lucid Creations
├── RJ Print
├── LastStopShop
└── Etc.
```

## Role of Each Section

### Shop
Main product hub for Auralis Digital.

### Partnered Creators
A curated area for trusted friends, family, artists, makers, POD creators, and 3D print creators connected to Auralis Digital. This is not intended as an open public marketplace.

### Lucid Creations
Auralis Digital's original art/design/product line, including apparel, blankets, accessories, custom visual products, and Printify/POD products.

### RJ Print
3D printed products and custom print-related items, including practical accessories, koozies, lamps, wallets, cases, and related creator-made objects.

### LastStopShop
A separate shop/category for broader products, POD items, curated items, and future expansion under its own dedicated repository.

### Etc.
Reserved for future shop branches, categories, or additional trusted creator sections.

## Build Intent

1. Save this structure in the Auralis Digital repo.
2. Keep Auralis Digital as the umbrella brand/site hub.
3. Keep Lucid Creations, RJ Print, and LastStopShop as separate product/shop identities under the Shop section.
4. Start dedicated development work in the `LastStopShop` repo after this structure is recorded.
