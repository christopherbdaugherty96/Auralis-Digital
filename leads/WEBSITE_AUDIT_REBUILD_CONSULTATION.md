# Website Audit / Rebuild Consultation

Status: offer template
Last updated: 2026-05-06

Use this offer when a prospect already has a website that Auralis Digital did not build.

This creates a paid, professional path before promising cleanup, rebuild, migration, or ongoing support for an unknown third-party site.

## Suggested Price

Starting range: $150-$300

Final price depends on site size, platform complexity, access issues, urgency, and how much written detail the client needs.

## Purpose

The purpose of this consultation is to review an existing third-party website before quoting larger work.

It helps identify:

- what platform or builder the site uses
- whether the client has domain, hosting, and admin access
- whether forms, links, mobile layout, and core pages work
- whether the site should be kept, rebuilt, migrated, or quoted as custom work
- whether ongoing care is reasonable or risky

## Good Fit

Offer this when:

- the client already has a website
- Auralis did not build the current website
- the client wants cleanup, refresh work, migration, or ongoing support
- the current platform, hosting, plugins, forms, SEO setup, or access situation is unclear
- the client wants a professional recommendation before spending more

## Not A Good Fit

Do not position this as:

- a full redesign
- a guaranteed fix
- a full SEO campaign
- a guaranteed ranking improvement
- unlimited troubleshooting
- a replacement for a full project quote
- cheap third-party cleanup

## Review Areas

The consultation may review:

- homepage clarity
- mobile usability
- contact path and form behavior
- navigation and key pages
- current platform or builder
- domain and hosting ownership/access
- visible technical risks
- basic SEO structure
- obvious trust gaps
- maintenance or migration risk

## Deliverable

Deliver a short written recommendation.

Recommended format:

```text
Current situation:

Main issues found:

Access/platform notes:

Recommended path:
- keep and improve
- rebuild
- migrate
- quote custom work

Suggested next step:
```

The deliverable does not need to be a long report. It should be clear enough for the client to understand the recommended path and for Auralis to quote the next step responsibly.

## Outcome Options

After the consultation, recommend one of:

### Keep And Improve

Use only when the current site is stable, accessible, and worth improving.

### Rebuild

Use when the current site has poor structure, weak mobile experience, unclear messaging, or too much inherited complexity.

### Migrate

Use when the current platform, hosting, or ownership setup is the main issue.

### Custom Quote

Use when the work is too specific or risky for standard pricing.

## Client-Friendly Explanation

Use language like:

> Since Auralis did not build the current site, I do not want to promise a cheap refresh before reviewing the setup. The audit/rebuild consultation gives you a clear recommendation first: keep it, rebuild it, migrate it, or quote custom work.

## Internal Rules

- Keep Website Refresh limited to Auralis-built sites only.
- Do not promise cheap third-party cleanup.
- Do not promise guaranteed SEO/ranking results.
- Do not offer unlimited updates or unlimited troubleshooting.
- Do not move a third-party site into managed hosting and care until the setup has been reviewed.
- Convert larger work into a separate quote.
